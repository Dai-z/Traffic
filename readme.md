How to install and use.

## Preinstall
### Linux
$ sudo apt-get install cmake qt

### Windows

install cmake 
install qt

use cmake-gui to build or use command(see "Install&Build")

##Install&Buid
open terminal(cmd in Windows), cd to /path/to/repository

enter command below
$ mkdir build
$ cd build
$ cmake ..
$ make


At last run "traffic" in build folder
